MediaPipe Machine Check

Run these three local demos on each machine. The goal is to confirm that the browser can load MediaPipe WASM, download the bundled model, open the camera, and keep the overlay responsive.

Pose Estimation

Full body pose tracking with optional hand landmarks.

Launch pose test
Face Mesh

Face landmark mesh, contours, iris points, and smoothing.

Launch face test
Gesture Recognition

Hand landmark tracking with the gesture recognizer task.

Launch hand test

Record For Each Demo

  1. Does the model load without an error?
  2. Does the browser request camera permission?
  3. Does the overlay follow the person smoothly?
  4. What FPS does the demo report after 10 seconds?

Run Locally

  • Start a local server from the repo root.
  • Open /mediapipe-lab/ from localhost.
  • Do not open the HTML file directly from Finder.
MediaPipe task models and WASM files are bundled in this folder so the exercise does not depend on a CDN during class.
